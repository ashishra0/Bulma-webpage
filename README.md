# Bulma CSS Framework

### Bulma is an open source framework based on Flexbox which aims at rapid website prototyping.

#### Installation

1) With npm
```bash
npm install bulma
```
2) With yarn
```bash
yarn add bulma
```
3) With bower
```bash
bower install bulma
```
4) CDN

https://cdnjs.com/libraries/bulma
